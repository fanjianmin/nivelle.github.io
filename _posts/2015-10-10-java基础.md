#### 奇数性

java对取余操作(%)的定义产生的后果。该操作符被定义为对所有的int数值a和所有的非零数值b,都满足下面恒等式：

```
(a/b)*b+(a%b)==a
```
与java的截尾整数正常操作符号相结合时，意味着：当取余操作返回一个非零的结果时，它与做操作数具有相同的正负符号。

正确做法：

```
public static bolean isOdd(int i){
    return i%2 !=0;
}

```
或者：

```
pulic static boolean isOdd(int i){
    return (i&1)!=0
}
```
